# HEW2018

HEW2018のプロジェクト

![](https://raw.githubusercontent.com/konojunya/HEW2018/master/screenshots/screen.png)

## Description

HAL Event Week 2018にて使った投票システムです。

**今回の構成**

- クライアント
  - Vue.js
  - axios
  - vue-router
  - localStorage
- サーバー
  - gin（Golang）
  - go-cache
  - gorm
- データベース
  - RDS（MySQL）
- インフラ
  - Docker
  - CircleCI
  - AWS（ECS + ALB + RDS + Route53）
